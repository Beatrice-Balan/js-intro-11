HOW do you create functions?

    1. use keyword "function" to declare a function
    2. Always give meaningful name
    3. decide if you will have arguments or not
    4. implement the body of the function: block of code will be executed each time you invoke your function

// Function declaration: 
function printHi() {
    console.log('Hi')
}

printHi();

2 types of functions/ methods you can create:
void method/ functions: they do Not return any value
return method-function: returns a value

RETURN TYPE FUNCTIONS:
    -> return a value or data when invoked

function multiply(num1, num2) {
    let result = num1 * num2;
    return result;
}
